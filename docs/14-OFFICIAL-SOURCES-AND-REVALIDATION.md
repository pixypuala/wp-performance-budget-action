# Official Sources and Revalidation

This blueprint was aligned to official documentation available on 2026-07-18. Platform details change; implementation must re-check these sources at project start, before each major milestone, and before release.

- **Core Web Vitals:** https://web.dev/articles/vitals
- **WordPress Developer Resources:** https://developer.wordpress.org/
- **GitHub Actions Security:** https://docs.github.com/en/actions/security-for-github-actions/security-guides/security-hardening-for-github-actions
- **Semantic Versioning:** https://semver.org/

## Revalidation checklist

- Current WordPress stable, previous supported and next beta/RC.
- Current PHP support/security status and WordPress compatibility guidance.
- Current block.json/theme.json/API schema versions.
- Current Interactivity/Abilities/Playground/WooCommerce API stability and deprecations where applicable.
- Current browser, Node/runtime and GitHub Actions supported versions.
- Current WCAG/WAI-ARIA guidance and known browser/assistive-technology behavior where applicable.
- Current dependency licenses, vulnerabilities and maintenance status.

Record date, reviewer and resulting ADR/dependency/matrix changes. Do not copy version claims from this document indefinitely.
